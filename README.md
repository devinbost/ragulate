# ragulate

A tool for evaluating RAG pipelines

![](logo_smaller.png)


## Installation

```sh
pip install ragulate
```

## Usage

```sh
ragulate -h

usage: ragulate [-h] {download-llamadataset} ...

RAGulate CLI tool.

options:
  -h, --help            show this help message and exit

commands:
    download-llamadataset
                        Download a llama-dataset
```
